WebGPU required

This game needs WebGPU, which isn't enabled in your browser yet.

Steps to enable it:

Chrome or Edge (easiest)
Update to version 113 or newer — WebGPU is on by default. Check your version at chrome://settings/help or edge://settings/help and click Update if offered.

Chromium (Linux)
1. Go to chrome://flags/#enable-unsafe-webgpu.
2. Set Unsafe WebGPU Support to Enabled.
3. Click Relaunch.
If it still doesn't work, launch Chromium from a terminal with:
chromium --enable-unsafe-webgpu --enable-features=Vulkan

Firefox
1. Type about:config in the address bar and press Enter.
2. Accept the warning.
3. Search for dom.webgpu.enabled.
4. Double-click it to set it to true.
5. Reload this page.

Safari 18+
1. Open Safari → Settings → Advanced and tick Show features for web developers.
2. Open Develop → Feature Flags.
3. Search for WebGPU and enable it.
4. Reload this page.
